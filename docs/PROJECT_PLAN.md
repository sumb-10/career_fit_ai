# PROJECT_PLAN.md — CareerFit AI 프로젝트 기획 초안



## 1. 문제 정의

취업 준비생은 수많은 채용 공고와 공모전 정보를 비교하면서,
자신의 전공과 보유 역량이 어떤 직무에 적합한지 판단하기 어렵다.
특히 어떤 역량을 새로 갖춰야하는지, 어떤 순서로 준비해야 수월한지 알기 어렵다.

CareerFit AI는 목표로 하는 직군, 직무 별로 요구되는 스킬을 데이터로 정리하고, 입력된 현재 나의 상태를 체계적으로 분석하여, 공고 데이터를 기반으로 부족한 역량 체크와 효과적인 준비 방향을 제안하는 AI 포트폴리오 코치 서비스이다.

## 2. 사용자 정의

- 대상: 취업과 공모전을 준비하는 대학 3·4학년 학생


- 사용 시나리오:

1. 사용자가 전공, 보유 스킬, 관심 직무를 입력한다.

2. CareerFit AI가 데이터를 검색해 역량 매칭 결과를 반환한다.

3. 어떤 데이터를 기반으로 답변했는지 출처를 함께 표시한다.



## 3. 데이터 계획

- 취업 공고 컬럼: [회사명, 직무명, 요구스킬, 우대사항, 지원자격, 업무내용, 마감일]

- 공모전 컬럼: [공모전명, 주최, 분야, 참가 자격, 요구 역량, 상금, 마감일]

- 데이터 출처: 강사 제공 목업 CSV (3일차)
